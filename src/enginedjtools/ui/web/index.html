      User Manual

      Getting Started

      Engine DJ Tools connects directly to your Engine DJ library database. Open a library by clicking OPEN LIBRARY in the top bar and selecting the folder that contains your Music database folder (the one with m.db inside).

      Collections, Crates & Playlists — What's the Difference?

      In Engine DJ, Crates and Playlists are stored in the same Playlist database table — they are structurally identical at the database level. The distinction is purely in how the Engine DJ application presents them to you in its UI.

      • Crates — unordered tag-like containers. Think of them as labels or folders. Great for genre, mood, or source-based grouping.
      • Playlists — ordered lists. The track order matters and is preserved.
      • Nesting — both crates and playlists support sub-folders using a parent/child relationship (parentListId).

      In Engine DJ Tools, all of these are referred to collectively as Collections.

      Can a track be in multiple collections? Yes. A track can belong to as many collections as you like simultaneously. The Collections column in the Tracks panel shows all collections a track belongs to.

      Collections Panel

      The Collections panel shows all your crates and playlists in a tree on the left. Click any collection to see the tracks it contains on the right.

      • Right-click a track — open the context menu to add the track to another collection, remove it, or play it.
      • Double-click a track — open the waveform player.
      • Multi-select — use Ctrl+Click to toggle individual tracks, Shift+Click to select a range. A selection bar appears at the bottom with ADD TO COLLECTION and CLEAR buttons.

      Tracks Panel

      The Tracks panel shows your full library organised by Label → Album. Expand a group on the left to see the tracks on the right.

      • ALL / UNCOLLECTED toggle — switch between showing every track or only tracks not yet added to any collection.
      • Quality filters — NO BPM, NO KEY, NO WAVEFORM — quickly find tracks that haven't been fully analysed by Engine DJ.
      • COLLECTIONS column — shows which collections each track belongs to (comma-separated). Empty means the track is uncollected.
      • Multi-select — same Ctrl+Click / Shift+Click behaviour as the Collections panel. Use the selection bar to bulk-add tracks to a collection.

      Backup Debugger

      Analyses your Engine DJ backup folder structure and flags any anomalies such as missing backup sets, corrupt archives, or out-of-sequence versions. Use this after any drive transfer to verify the backup completed cleanly.

      Manual Backup

      Creates a dated ZIP snapshot of your Engine DJ database files. Keep these backups before any bulk edits. Backups are stored in a Backups sub-folder inside your library folder.

      Library Stats

      Shows a summary of your library health: total track count, how many tracks have BPM, Key, and Waveform data, plus database size and schema version. Use this to gauge how much analysis work Engine DJ still needs to do.
